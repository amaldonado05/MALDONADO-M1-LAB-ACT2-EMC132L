# M1 - Game Programming 2: Shooting Player

A simple 3D Unity first-person shooter demonstrating player control, shooting mechanics, enemy interaction, and scoring using C# scripting.

## Features
- FPS-style player movement and camera look
- Gun that shoots bullets with sound effects  
- Four enemies with different point values  
- Enemies respawn after being destroyed  
- Real-time score displayed on screen

## How to Play
1. Move using WASD and look around with your mouse.  
2. Left-click to shoot bullets.  
3. Hit enemies to destroy them and earn points.  
4. Enemies respawn after a short delay — keep shooting!

## Project Overview
- Demonstrates shooting logic, prefabs, triggers, and scoring in Unity.  
- Uses C# scripts for player control, physics, and sound effects.  
- Great introduction to first-person game mechanics.
